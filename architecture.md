# アーキテクチャ図

以下は、CloudFormationテンプレートによって作成されるアーキテクチャの概要です。

```
+------------------------------------------+
|                                          |
|  VPC (10.0.0.0/16)                       |
|                                          |
|  +--------------------------------------+|
|  |                                      ||
|  |  Public Subnet (10.0.1.0/24)         ||
|  |                                      ||
|  |  +---------------+                   ||
|  |  |               |                   ||
|  |  |  EC2 Instance |                   ||
|  |  |               |                   ||
|  |  +-------+-------+                   ||
|  |          |                           ||
|  +----------|--------------------------+|
|             |                           |
|  +----------v---------+                 |
|  |                    |                 |
|  | Internet Gateway   |                 |
|  |                    |                 |
|  +--------------------+                 |
|                                          |
+------------------------------------------+
           |
           |
           v
      Internet
```

## 説明

1. **VPC** (10.0.0.0/16) - 開発環境用の仮想ネットワーク
2. **パブリックサブネット** (10.0.1.0/24) - EC2インスタンスが配置されるサブネット
3. **EC2インスタンス** - 開発作業用のコンピューティングリソース
4. **インターネットゲートウェイ** - VPCとインターネット間の通信を可能にする
5. **ルートテーブル** (図には表示されていません) - トラフィックの経路を定義
6. **セキュリティグループ** (図には表示されていません) - EC2インスタンスへのSSHアクセスを制御
